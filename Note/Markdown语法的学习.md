# Markdown语法的学习  

## 1.标题语法

在行首添加**#**即可使该行成为标题，有几个**#**就是第几级标题，最次级是第六级。  

> *注意：**#**后要添加一个**空格**，否则系统无法识别标题指令。*  

## 2.段落语法  

段落行首不可以加**空格**或**Tab**。  

> *注意：段落指文字说明段落。*  

## 3.换行语法  

一般地，换行时应在行尾添加**2个或以上空格**。  

> *教程解释部分编译器能够接受不加空格，但为确保执行不出错，最好养成加空格的习惯。*  

## 4.强调语法  

**加粗**：在要加粗的内容两侧分别加上**2个*或下划线**。  

*斜体*：在要斜体的内容两侧分别加上_1个*或下划线_。  

***加粗并斜体***：在要加粗并斜体的内容两侧分别加上___3个*或下划线___。  

> *这里有疑问，星号与下划线出现在被强调的内容中时会打断强调，不知道如何解决。*  

> _注意：行中两旁无空格时强调语法最好使用*。_

 ## 5.引用语法  

在行首添加 **>** 可使该行渲染引用效果，块引用可多行连续，**>** 可叠加使用，在块引用中嵌套次级引用。  

> *注意：引用中可以添加部分其他语法元素，例如斜体。*  

> *另：创作中发现，若块引用行尾换行后自动多行连续，可再次点击enter去除引用渲染效果。

## 6.列表语法  

1.有序列表  

​	有序列表在行首添加数字与英文句点（**.**），以数字**1**起始，后面数字可不遵循顺序。  

2.无序列表

​	在行首添加*****，**-**或**+**，后接一空格。

3.列表嵌套  

​	链表内可嵌套多种元素，嵌套时要在行首基于嵌套语法本身行首缩进格式再加**4个空格或1个Tab**。  

​	目前已知可嵌套元素有：

	-  文字说明段落
	-  块引用(后面笔记中发现无法进行引用渲染，不知道为什么)
	-  代码块  
	-  图片  
	-  列表 

## 7.代码语法  

若将句中某部分内容表达为代码，应在内容前后分别添加**1个反引号（`）**。  

若要在笔记中插入代码块，需要在每一行首都缩进**4个空格或1个Tab**。  

例：  

​	int main()

​	{

​	printf("Hello world!\n");

​	return 0;

​	}  

> *注意：若代码中存在反引号，则首尾反引号数量变为2个。*  

## 8.分隔线语法  

在一行中单独使用**3个或以上 *，- 或 _ 时可形成分隔线。  

> *注意：分隔线所在行除指定的语法符号外不可再添加其他内容，分隔线前后应各空出一行。*

## 9.链接语法  

超链接格式：[超链接显示名] 后接 (超链接地址 "超链接title")。  

例：本篇笔记学习素材来自[Markdown教程](https://markdown.com.cn/)。  

> *注意："超链接title"为非必需项，鼠标光标悬停在超链接上所出现的文字就是超链接title，且超链接地址与超链接title中间要隔**1个空格**。*  

使用**尖括号（<>）**可以直接将网址（URL）或邮箱地址变为可点击的链接。  

例：本篇笔记学习素材来自<https://markdown.com.cn/>。  

若想要强调链接，可直接使用强调语法，若想要将超链接显示名渲染为代码格式，应在中括号内超链接显示名前后添加反括号（`）。  

引用类型链接类似索引，分为两部分：  

1.我称为”导航部分“，包含**2个[]**，第一个内存放链接显示名，第二个内存放一个标签，标签名可以是字母，数字，空格或标点符号。  

​	>*注意：2个[]中间最好添加1个空格。*  

2.我称为”解释部分“，包含：  

	- [标签名]  
	- 1个冒号（:）和至少1个空格  

 - 超链接地址与超链接title  

​	>*注意：此处超链接地址可选择性添加尖括号（<>），超链接title前后添加 " "，' ' 或 () 。*  

> *注意：若网址（URL）中含**空格**，最好将其替换为**%20**。*  

## 10.图片语法  

插入图片时应先输入一个感叹号（!），后接 [图片名称（替代文本，不会显示，不重要）]，与超链接格式相似，图片名称后是 (图片链接 "图片title")。  

链接图片即将上述格式作为超链接显示名放在[]中，其他与链接语法相同。  

> *注意："图片title"为非必需项，鼠标光标悬停在图片上所出现的文字就是图片title。
